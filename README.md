# NAN 2026 출품작 (워킹 타이틀)

**움직일 때만 시간이 흐르는** 서바이벌 액션. NAN 2026 (NHN Game × AI Hackathon) 사전 과제 출품작.

## 플레이

- 웹 (링크 클릭 즉시 플레이): **https://biddan606.github.io/nan2026-game/**
- 조작: WASD/방향키 이동, 스페이스 대시 (예정). 공격은 자동.
- 멈추면 세상도 거의 멈춘다. 움직이는 순간 적도, 탄막도, 시간도 함께 달린다.

## 로컬 실행

```bash
npm install
npm run dev
```
