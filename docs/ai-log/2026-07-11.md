핵심 발견: 우리 트위스트는 slither.io 부스트(가속 대가로 질량 소모)와 동형 구조인데 보상 축이 비어 있었음. 콤보-스코어가 그 축을 채움.

### 확정 설계 — 시간 경제

- 콤보: 킬 연속 배율, **실시간 감소** (시간을 멈춰도 식음 — 멈춤이 콤보까지 보존하면 딜레마 소멸)
- 집중 게이지: 정지 중 충전, 대시(무적 프레임)가 소모
- 적 합체: 일반 적 둘이 뭉치면 엘리트 — 카이팅·장기 정지의 대가 (사용자 아이디어 채택)
- 적 지능화는 "예측 조준 1종"으로 한정 (전역 경로 탐색은 성능·공정성 문제로 기각)

### 구현 (프로토타입 v2)

자동공격(최근접 조준, 발사 타이머가 Clock timeScale에 걸려 시간 흐름과 자동 연동), 콤보 HUD, 집중 게이지 3칸, 스페이스 대시, 적 합체(엘리트 동시 4마리 상한, 스폰 직후 2.5초 합체 금지), 사망 화면 + R/스페이스 즉시 재시작, localStorage 최고 점수.

## 플레이테스트 피드백 반영 — 불릿타임 자원화 (v3)

사용자 플레이 피드백: "멈춰있고 싶어서 멈추는데 게임이 멈춰버림" — 정지 시 자동 슬로우모가 전술적 정지를 일시정지로 만드는 퇴화 루프 발견. Claude가 세 가지 대안(제거/자원화/완화)을 트레이드오프와 함께 제시, 사용자가 자원화 선택 (ADR-0003).

- 세상은 항상 정상 속도. 슬로우모는 Shift 홀드 불릿타임(게이지 1칸당 1.5초)으로만
- 게이지 충전 = 완전 정지 상태에서만 — 적이 정상 속도로 다가오는 동안의 위험한 투자
- ESC 일시정지 분리 (물리 + 타이머 정지, 오버레이)
- 콤보 실시간 감소 유지 — 불릿타임 중에도 식음

## 난이도 디렉터 (v4)

사용자 피드백: "가만히 있어도 안 끝남" — 고정 800ms 스폰이 자동공격 DPS에 밀려 밀도가 영원히 바닥. 웹 리서치로 슈팅 공식 3개 수집 후 조합:

1. **최소 밀도 쿼터** (Vampire Survivors): 화면 최소 적 수 = 5 + t/15초, 미달 시 즉시 충원 — 킬 속도와 무관하게 밀도 바닥 보장
2. **sqrt/선형 곡선** (abagames 소형 게임 설계론): 적 속도 선형 증가 (상한 1.6배), 스폰 간격 선형 단축
3. **피크·밸리** (Left 4 Dead AI 디렉터): 45초마다 최소 밀도 60%를 링 스폰(피크), 직후 10초 스폰 정지(밸리)

난이도 시계는 게임 시간 기준 — 불릿타임이 난이도 진행도 늦춰서 "시간=자원" 정체성 강화. HUD에 생존 시간 표시 추가.

## 성장 시스템 (v5)

사용자 피드백: "초반 60초 지루, 서 있게 됨. 초반엔 약하고 킬로 성장하게" — 뱀서 코어인 XP·레벨업이 빠져 있었음.

- 시작 스탯 하향: 발사 간격 350ms → 900ms, 단발 — 초반은 도망·수집이 본업
- 킬 → XP 보석 드랍 (엘리트 3개), 자석 반경 내 흡인, 수집 시 XP
- 레벨업 → 시간 정지 + 3택 카드 (1/2/3 키 or 클릭): 공속/갈래(부채꼴 멀티샷)/피해/신속/집중(게이지 최대)/자석 — 계획했던 업그레이드 6종
- XP 곡선: 5 + (레벨−1)×3, 넘친 XP 이월로 연속 레벨업 지원
