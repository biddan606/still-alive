# 불릿타임을 수동 자원으로 전환 — 정지 시 자동 슬로우모 폐기

초기 코어 규칙은 "플레이어가 멈추면 세상이 느려진다"(SUPERHOT식 수동태)였다. 플레이테스트에서 이 규칙이 퇴화 루프를 만들었다: 멈춤 = 세상 정지 = 위험 없이 게이지만 차는 사실상의 일시정지 버튼이 되어, 전술로서의 "서 있기"가 성립하지 않았다. 이를 뒤집어 세상은 항상 정상 속도로 흐르게 하고, 슬로우모는 집중 게이지를 소모하는 능동 능력(Shift 홀드 불릿타임)으로 옮겼다. 게이지는 완전히 멈춰 서 있는 동안만 차므로 충전 자체가 위험한 투자가 된다. 시간 조작이라는 차별점은 유지하되, 자동 규칙에서 자원 관리로 진화시킨 결정. 일시정지는 ESC로 분리했다.

추가 (2026-07-16): 두 소비처의 역할을 경제로 갈랐다. 대시는 반 칸짜리 기본기(짧은 락아웃으로 무적 체인만 금지), 불릿타임은 발동 시 1칸을 선불하는 큰 커밋으로. 선불 1칸의 값어치(0.8초)는 발동 즉시 선불 시간으로 확보되므로 끝까지 태우면 손해가 없고, 짧게 켰다 끄면 남은 선불을 버린다 — 부분 사용의 유연성 우위로 불릿타임이 대시를 지배하던 문제와, 게이지 바닥 근처에서 켜짐/꺼짐을 반복해 공짜 슬로우모를 얻던 익스플로잇을 함께 막는다.
